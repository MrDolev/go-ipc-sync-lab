# go-ipc-sync-lab

A Go-based laboratory for exploring Inter-Process Communication (IPC) and synchronization patterns using goroutines and channels.

## Overview

This project demonstrates fundamental concepts of concurrent programming including:
- **Mutex** - Mutual exclusion for critical sections
- **Semaphores** - Resource counting and access control
- **Producer-Consumer** - Classic concurrent design pattern

## Background

This project originated from project work on my bachelor's degree in **Fondamenti di Informatica** (Fundamentals of Computer Science), where I contributed to improving the theoretical descriptions of IPC mechanisms on the Italian Wikipedia page for [Comunicazione tra processi](https://it.wikipedia.org/wiki/Comunicazione_tra_processi) (Inter-Process Communication).

The goal is to provide practical Go implementations of these theoretical concepts, making them easier to understand through working examples.

## Features

- Practical demonstrations of synchronization primitives
- Real-world examples of common concurrency problems
- Educational code with clear documentation
- Race condition detection using Go's race detector
